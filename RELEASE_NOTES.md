# Mineradio 1.2.0 Modified - Kugou Support

这是基于 [XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio) 的非官方修改版。

## 修改内容

- 新增酷狗音乐单曲搜索。
- 新增酷狗音乐歌单搜索。
- 支持 QQ 音乐 / 酷狗音乐歌单搜索结果展示。
- 支持酷狗歌单加载到播放队列。
- 接入酷狗播放地址、歌词、登录状态与网页登录入口。
- 调整前端搜索入口和多平台登录状态展示。

## 下载和使用

下载 `Mineradio-1.2.0-kugou-portable.zip`，解压后运行 `Mineradio.exe`。
